FAQ

Soalan lazim.

Adakah saya perlu tahu coding untuk guna EZMeta? +
Tidak langsung. Anda hanya perlu bagi Ad Account ID dan setup Telegram. Kami yang handle semua technical setup.
EZMeta akan pause atau ubah ads saya secara automatik? +
Tidak — EZMeta dalam mod "cadangan" secara default. AI akan recommend actions, tapi anda yang approve. Kami tak akan ubah campaigns anda tanpa kebenaran.
Berapa lama untuk setup? +
Kurang dari 25 minit dari mula sampai laporan pertama masuk Telegram anda.
Adakah data ads saya selamat? +
Ya. Kami guna Meta official API yang sama digunakan oleh tools seperti Hootsuite dan AdEspresso. Data anda tidak dikongsi dengan mana-mana pihak ketiga.
Boleh cancel bila-bila masa? +
Ya, boleh cancel bila-bila masa tanpa penalty. Tiada kontrak jangka panjang.
